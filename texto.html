No un programa gigante ni ejemplos demasiado cortos. --- # 1. ¿Qué es un método? Un método es un bloque de código que realiza una tarea específica y puede ser reutilizado varias veces dentro de un programa. ### Sintaxis ```java public static void nombreMetodo(){ } ``` ### Ejemplo ```java public class Main { public static void bienvenida(){ System.out.println("Bienvenido al sistema"); } public static void main(String[] args){ bienvenida(); } } ``` Salida: ```text Bienvenido al sistema ``` --- # 2. Métodos sin parámetros Son métodos que no necesitan recibir información. ```java public class Main { public static void mostrarMensaje(){ System.out.println("Programación en Java"); } public static void main(String[] args){ mostrarMensaje(); mostrarMensaje(); mostrarMensaje(); } } ``` Salida: ```text Programación en Java Programación en Java Programación en Java ``` --- # 3. Métodos con parámetros Los parámetros permiten enviar información al método. ```java public class Main { public static void saludar(String nombre){ System.out.println("Hola " + nombre); } public static void main(String[] args){ saludar("Carlos"); saludar("María"); saludar("Pedro"); } } ``` Salida: ```text Hola Carlos Hola María Hola Pedro ``` --- # 4. Métodos con varios parámetros ```java public class Main { public static void mostrarDatos(String nombre, int edad){ System.out.println("Nombre: " + nombre); System.out.println("Edad: " + edad); } public static void main(String[] args){ mostrarDatos("Juan",20); } } ``` --- # 5. Métodos con retorno (return) Devuelven un valor. ```java public class Main { public static int sumar(int a,int b){ return a+b; } public static void main(String[] args){ int resultado; resultado = sumar(10,5); System.out.println(resultado); } } ``` Salida: ```text 15 ``` --- # 6. Métodos que retornan boolean Muy utilizados en condiciones. ```java public class Main { public static boolean esPar(int numero){ return numero % 2 == 0; } public static void main(String[] args){ System.out.println(esPar(8)); System.out.println(esPar(7)); } } ``` Salida: ```text true false ``` --- # 7. Entrada de datos con Scanner Permite que el usuario ingrese información. ```java import java.util.Scanner; public class Main { public static void main(String[] args){ Scanner sc = new Scanner(System.in); System.out.print("Ingrese su nombre: "); String nombre = sc.nextLine(); System.out.println("Bienvenido " + nombre); } } ``` --- # 8. Método que recibe datos del usuario ```java import java.util.Scanner; public class Main { public static void saludar(String nombre){ System.out.println("Hola " + nombre); } public static void main(String[] args){ Scanner sc = new Scanner(System.in); System.out.print("Ingrese nombre: "); String nombre = sc.nextLine(); saludar(nombre); } } ``` --- # 9. Método que devuelve datos ingresados ```java import java.util.Scanner; public class Main { public static int pedirNumero(){ Scanner sc = new Scanner(System.in); System.out.print("Ingrese un número: "); return sc.nextInt(); } public static void main(String[] args){ int numero = pedirNumero(); System.out.println(numero); } } ``` --- # 10. Condicional If Permite tomar decisiones. ```java import java.util.Scanner; public class Main { public static void main(String[] args){ Scanner sc = new Scanner(System.in); System.out.print("Ingrese edad: "); int edad = sc.nextInt(); if(edad >= 18){ System.out.println("Mayor de edad"); } } } ``` --- # 11. If - Else ```java import java.util.Scanner; public class Main { public static void main(String[] args){ Scanner sc = new Scanner(System.in); System.out.print("Ingrese edad: "); int edad = sc.nextInt(); if(edad >= 18){ System.out.println("Mayor"); }else{ System.out.println("Menor"); } } } ``` --- # 12. If Else If ```java import java.util.Scanner; public class Main { public static void main(String[] args){ Scanner sc = new Scanner(System.in); System.out.print("Ingrese nota: "); int nota = sc.nextInt(); if(nota >= 18){ System.out.println("Excelente"); }else if(nota >= 14){ System.out.println("Bueno"); }else if(nota >= 11){ System.out.println("Aprobado"); }else{ System.out.println("Desaprobado"); } } } ``` --- # 13. Switch Ideal para menús. ```java import java.util.Scanner; public class Main { public static void main(String[] args){ Scanner sc = new Scanner(System.in); System.out.println("1. Nuevo"); System.out.println("2. Modificar"); System.out.println("3. Eliminar"); int opcion = sc.nextInt(); switch(opcion){ case 1: System.out.println("Nuevo"); break; case 2: System.out.println("Modificar"); break; case 3: System.out.println("Eliminar"); break; default: System.out.println("Opción inválida"); } } } ``` --- # 14. Bucle For Cuando sabemos cuántas veces repetir. ```java public class Main { public static void main(String[] args){ for(int i=1;i<=10;i++){ System.out.println(i); } } } ``` --- # 15. For con métodos ```java public class Main { public static void saludar(){ System.out.println("Hola"); } public static void main(String[] args){ for(int i=1;i<=5;i++){ saludar(); } } } ``` --- # 16. While Mientras la condición sea verdadera. ```java public class Main { public static void main(String[] args){ int numero = 1; while(numero <= 5){ System.out.println(numero); numero++; } } } ``` --- # 17. Do While Ejecuta al menos una vez. ```java public class Main { public static void main(String[] args){ int numero = 1; do{ System.out.println(numero); numero++; }while(numero <= 5); } } ``` --- # 18. Método + If ```java public class Main { public static boolean esPositivo(int numero){ return numero > 0; } public static void main(String[] args){ if(esPositivo(10)){ System.out.println("Positivo"); } } } ``` --- # 19. Método + For Tabla de multiplicar. ```java public class Main { public static void tabla(int numero){ for(int i=1;i<=10;i++){ System.out.println( numero + " x " + i + " = " + (numero*i)); } } public static void main(String[] args){ tabla(5); } } ``` --- # 20. Método + Scanner + If Ejercicio completo. ```java import java.util.Scanner; public class Main { public static boolean esMayorEdad(int edad){ return edad >= 18; } public static void main(String[] args){ Scanner sc = new Scanner(System.in); System.out.print("Ingrese edad: "); int edad = sc.nextInt(); if(esMayorEdad(edad)){ System.out.println("Mayor de edad"); }else{ System.out.println("Menor de edad"); } } } ``` --- # 21. Método + Scanner + For ```java import java.util.Scanner; public class Main { public static void contar(int numero){ for(int i=1;i<=numero;i++){ System.out.println(i); } } public static void main(String[] args){ Scanner sc = new Scanner(System.in); System.out.print("Ingrese número: "); int numero = sc.nextInt(); contar(numero); } } ``` --- # Conceptos que debes saber explicar | Concepto | Explicación | | -------------- | ------------------------------------------- | | Método | Bloque de código reutilizable | | Parámetro | Variable que recibe un método | | Argumento | Valor enviado al método | | Return | Devuelve un resultado | | Scanner | Permite ingresar datos | | If | Ejecuta si una condición es verdadera | | Else | Se ejecuta cuando el if es falso | | Switch | Selecciona entre varias opciones | | For | Repetición con cantidad conocida | | While | Repite mientras una condición sea verdadera | | Do While | Ejecuta primero y luego verifica | | Boolean | Solo puede ser true o false | | Modularización | Dividir el programa en métodos | Con estos 21 puntos tienes prácticamente una unidad completa de **Métodos, Entrada de Datos, Condicionales y Bucles en Java**, organizada de forma progresiva y fácil de exponer.
